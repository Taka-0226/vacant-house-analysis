# 全国市区町村の空き家率分析

## 概要

本プロジェクトでは、全国の市区町村を対象に、住宅・土地統計調査における「空き家率」と地域特性の関係を分析しました。

人口構造、人口動態、住宅ストック、財政力、居住密度などの公開統計データを用いて、その他空き家率に関連する要因を確認し、さらに自治体をタイプ分類することで、タイプ別の空き家対策の方向性を検討しました。

## 分析の目的

近年、人口減少や高齢化に伴い、空き家の増加が地域課題となっています。  
本分析では、単に空き家率を予測するだけでなく、どのような地域特性を持つ自治体で空き家率が高くなりやすいのかを把握し、自治体タイプごとの施策提案につなげることを目的としました。

## 使用データ

本分析では、以下の公開統計データを使用しました。

- 住宅・土地統計調査
  - その他空き家率、持ち家率、建築時期別住宅割合の算出に使用
- 人口統計
  - 総人口、65歳以上人口、高齢化率の算出に使用
- 住民基本台帳人口
  - 人口増減率の算出に使用
- 面積データ
  - 総面積、可住地面積、人口密度、可住地人口密度の算出に使用
- 市町村税課税状況等の調
  - 納税義務者1人当たり課税対象所得の取得に使用
- 地方公共団体の主要財政指標一覧
  - 財政力指数の取得に使用

※ 正式な取得元・URLは整理中です。

## 目的変数

本分析では、住宅・土地統計調査における「賃貸・売却用及び二次的住宅を除く空き家」を総住宅数で除した値を「その他空き家率」として定義しました。

## 分析手法

主に以下の分析を行いました。

- 基礎集計・可視化
- 相関分析
- 多重共線性の確認（VIF）
- 重回帰分析（OLS）
- 標準化OLSによる影響度比較
- LinearRegressionによる予測性能確認
- KMeansによる自治体クラスタリング
- 自治体タイプ別の施策提案

## 主な結果

相関分析および重回帰分析の結果、空き家率は特に高齢化率および1980年以前建築住宅割合と強く関連していることが確認されました。

また、可住地人口密度や財政力指数はその他空き家率と負の関係を示しました。これは、居住密度が高く、財政基盤が比較的強い自治体では、住宅需要や行政対応力が一定程度保たれ、その他空き家率が低くなりやすい可能性を示しています。

LinearRegressionによる予測性能の確認では、テストデータにおいて以下の結果となりました。

- R²：0.667
- RMSE：2.67
- MAE：1.93

公開統計のみを用いたモデルとして、一定の予測性能が確認されました。

## 自治体タイプ分類

KMeansにより、自治体を以下の4タイプに分類しました。

| 自治体タイプ | 主な特徴 | 平均空き家率 |
|---|---|---|
| 都市高密度・低空き家型 | 高密度、低高齢化、人口維持・増加、老朽住宅割合が低い | 3.07% |
| 高齢化・人口減少・老朽住宅型 | 高齢化、人口減少、老朽住宅割合が高く、財政力が低い | 14.37% |
| 地方中間・住宅ストック課題型 | 各指標は平均に近いが、都市型より空き家率が高い | 8.14% |
| 広域都市・郊外安定型 | 財政力が高く、人口動態が比較的安定している | 4.53% |

クラスタリングには空き家率を使用していませんが、タイプ別の平均空き家率には明確な差が見られました。

## 施策提案

分析結果を踏まえ、自治体タイプごとに以下のような施策の方向性を整理しました。

| 自治体タイプ | 施策の方向性 |
|---|---|
| 都市高密度・低空き家型 | 中古住宅流通の促進、リノベーション支援、空き家化前の早期相談 |
| 高齢化・人口減少・老朽住宅型 | 相続相談、所有者把握、空き家管理支援、老朽危険家屋の除却補助 |
| 地方中間・住宅ストック課題型 | 空き家バンクの強化、移住・定住支援、リフォーム補助 |
| 広域都市・郊外安定型 | 予防的な空き家相談、郊外住宅地の再生、中古住宅流通支援 |
